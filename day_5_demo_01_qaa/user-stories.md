# User Stories

## Events

- As a stable manager, I can view a list of all upcoming events so that I know what is scheduled.
- As a stable manager, I can filter events by category (show, trail, clinic, grooming) so that I can find relevant events quickly.
- As a stable manager, I can see how many spots remain on each event so that I can plan staff deployment.

## Staff

- As a stable manager, I can view all active staff members and their roles so that I know who is available.
- As a stable manager, I can register a new staff member by entering their name, email, and role so that they are added to the system.
- As a stable manager, I can see how many events each staff member has attended so that I can recognise experienced contributors.

## Dashboard

- As a stable manager, I can see key statistics at a glance (total events, total staff, open spots, spots filled) so that I have an overview without opening each section.
