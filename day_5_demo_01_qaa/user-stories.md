# User Stories

## Events

- As a staff coordinator, I can view a list of all upcoming events so that I know what is scheduled.
- As a coordinator, I can filter events by category (fundraising, awareness, community, training) so that I can find relevant opportunities quickly.
- As a coordinator, I can see how many spots remain on each event so that I can plan volunteer deployment.

## Volunteers

- As a coordinator, I can view all active volunteers and their roles so that I know who is available.
- As a coordinator, I can register a new volunteer by entering their name, email, and role so that they are added to the system.
- As a coordinator, I can see how many events each volunteer has attended so that I can recognise experienced contributors.

## Dashboard

- As a coordinator, I can see key statistics at a glance (total events, total volunteers, open spots, spots filled) so that I have an overview without opening each section.
